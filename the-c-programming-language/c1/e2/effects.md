Effects of changing \\c character in printf statement:
    -\\a: Outputs without newline
    -\\b: Ouputs "Hello Worl"
    -\\c,d: Unknown escape sequence warning
    -\\e: Outputs without newline
    -\\f: Outputs with newline, except command prompt aligned with end of "Hello World"
    -\\g,h,i,j,k,l,m: Unknown
    -\\o: Expected '{' after \\o
    -\\p,q: Unknown
    -\\r: Compiles, no output
    -\\s: Unknown
    -\\t: Normal output with tab and no newline
    -\\u: Unknown
    -\\v: Same as \\f
    -\\w: Unknown
    -\\x: Expected hex digits after \\x
    -\\y,z: Unknown
    -\\0-9: Outputs without newline
